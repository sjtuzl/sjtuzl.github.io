<p>在EclipseCon 2006上，有一个主题介绍叫做“<a href="http://www.eclipsecon.org/2006/Sub.do?id=62">Top ten great, strange or weird open source plug-ins and RCP</a>”它列举出10个最佳Eclipse plugin & RCP程序，分别是：</p>
<p>10. regatta workbench - <a href="http://rowtown.org/regattaworkbench/">http://rowtown.org/regattaworkbench/</a></p>
<p>9.&nbsp; Xray a profiler for eclipse plugins - <a href="http://eclipsefaq.org/chris/xray/index.html">http://eclipsefaq.org/chris/xray/index.html</a></p>
<p>8. Rsswol: your next rss reader - <a href="http://rssowl.org">http://rssowl.org</a></p>
<p>7. jfire: run your business on Eclipse - <a href="http://jfire.org/">http://jfire.org/</a></p>
<p>6. abbot: test your UI like SAP, Rational and BEA do it! - <a href="http://sourceforge.net/projects/abbot">http://sourceforge.net/projects/abbot</a></p>
<p>5. amateras: xml, struts, html, jsf, uml, wtp for resin, hsqldb - <a href="http://amateras.sourceforge.jp/">http://amateras.sourceforge.jp/</a></p>
<p>4. MYLAR: Jazz Up your Eclipse now! - <a href="http://www.eclipse.org/mylar/">http://www.eclipse.org/mylar/</a></p>
<p>3. Incredible UI goodies -&nbsp;<a href="http://andrei.gmxhome.de/anyedit/">http://andrei.gmxhome.de/anyedit/</a>, <a href="https://eclipseutilplugins.dev.java.net/">https://eclipseutilplugins.dev.java.net/</a></p>
<p>2. azureus: the best bittorrent client, a great SWT UI - <a href="http://azureus.sourceforge.net/">http://azureus.sourceforge.net/</a></p>
<p>1. open source mashup: eZing Builder - <a href="http://sourceforge.net/projects/ezingbuilder">http://sourceforge.net/projects/ezingbuilder</a>, <a href="http://www.fckeditor.net/">http://www.fckeditor.net/</a></p>
