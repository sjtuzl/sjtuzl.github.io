<p>Sun open source Java under GPL v2 license (<a href="https://openjdk.dev.java.net/">openJDK</a>)是本周一最重要的事情。我阅读了Sun自己有关此事件的几乎所有报道，包括这份“<a href="http://www.sun.com/aboutsun/media/presskits/2006-1113/quotesheet.pdf">来自世界各地的声音</a>”（里面包括中科院和中国开源联盟的表态），仔细看了<a href="http://www.theserverside.com/news/thread.tss?thread_id=43046">TSS</a>, <a href="http://www.javalobby.org/java/forums/t84244.html">Javabobby</a>等处的群众留言。IBM也在当日发表了一个相对平静的<a href="http://news.com.com/2061-10795_3-6134853.html">声明</a>，阐述了自己的态度。至于Open source Java与Harmony的关系及影响，可以参考杨普的<a href="http://blog.csdn.net/paulex/archive/2006/11/14/1382743.aspx">博客文章</a>。这块石头入水的效力，或许在未来半年后开始显现。</p>
<p>周二，就是今天，IBM中国召开全体员工大会，IBM CEO Sam Palmisano在北京亲临现场。特邀嘉宾，<a href="http://www.amazon.com/World-Flat-Updated-Expanded-Twenty-first/dp/0374292795/sr=8-1/qid=1163511903/ref=pd_bbs_sr_1/104-2504121-6634359?ie=UTF8&s=books">The World is Flat</a>的作者Tom Friedman的精彩演讲，让人折服。我想所有听了他演讲的人都记住了这句话："What could be done, will be done. The last question: is it by you or to you?"
</p>
