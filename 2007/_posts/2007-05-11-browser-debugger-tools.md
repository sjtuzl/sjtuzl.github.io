<p>向任何人兜售浏览器的toolbar都需要很大的勇气，出于对长期与dojo之类Web client端代码打交道的程序员们的关心，我准备推荐下面几个浏览器插件，which可以大大的提高生产效率：</p>
<p>1. <a href="http://livehttpheaders.mozdev.org/">LiveHTTPHeader </a>(Firefox);</p>
<p>2. <a href="https://addons.mozilla.org/firefox/addon/60">Web Developer</a> (Firefox);</p>
<p>3. <a href="https://addons.mozilla.org/en-US/firefox/addon/1843">Firebug</a> (Firefox)</p>
<p>4. <a href="http://www.blunck.se/iehttpheaders/iehttpheaders.html">ieHTTPheaders</a> (IE)</p>
<p>5. <a href="http://www.microsoft.com/downloads/details.aspx?FamilyId=E59C3964-672D-4511-BB3E-2D5E1DB91038&displaylang=en">IE Developer Toolbar</a> (IE)</p>
<p>其中IE Developer Toolbar是MS刚刚释出的新货。
