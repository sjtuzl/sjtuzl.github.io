<p>今天IBM第一款企业社会软件包，Lotus Connections 1.0正式发布。Lotus Connections集成了企业员工名录、社区、博客、社会书签和活动管理，是IBM迈进social computing领域的第一个正式产品。</p>
<p>安装Lotus Connections需要WebSphere Application Server 6.1, DB2 9.2或者Oralce数据库，LDAP服务器和Tivoli Directory Integrator。</p>
<p>在过去的100天，我个人在Connections开发工作中的最直接的成果是修复了博客模块里的74个bug。</p>
<p>Lotus Connections 1.0.1目前正在开发中，主要是加入了多语言支持，包括英文、德文、法文、意大利文、西班牙文、葡萄牙文、简体中文、繁体中文、日文、韩文。</p>
