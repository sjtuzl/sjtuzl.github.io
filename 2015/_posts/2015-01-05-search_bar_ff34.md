<p>The latest Firefox 34 shipped with a revised search bar design. To be honest, I don't like it because I need to change the default search engine from time to time and do not want to click my preferred search engine every time I search.</p>
<p>Lucky it's possible to restore the previous design by modifying the Firefox configuration - to change '<strong><em>browser.search.showOneOffButtons</em></strong>' to <em><strong>false</strong> </em>(details - <a href="http://forums.mozillazine.org/viewtopic.php?f=23&amp;t=2890823">http://forums.mozillazine.org/viewtopic.php?f=23&amp;t=2890823</a>)</p>
