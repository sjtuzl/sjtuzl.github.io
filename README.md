# Ling Zhang's Blog — sjtuzl.github.io

Static blog on GitHub Pages, migrated from the WordPress site at
`zhangling.org/blog`. Built with Jekyll and deployed via GitHub Actions.

## Structure

| Path | Purpose |
|------|---------|
| `_layouts/`, `_includes/` | Global page chrome (header, nav, footer, comments). **Restyle the whole blog by editing these + `assets/css/style.css` only.** |
| `assets/css/style.css` | Single global stylesheet (Twenty Twenty look). |
| `assets/media/` | Migrated images & attachments (`uploads/`, `photos/`, `files/`, `special/`, `images/`). |
| `<year>/_posts/*.md` | Posts, organised by year. Each sets `permalink: /YYYYMMDD/slug.html`. |
| `index.html` | Paginated home. |
| `archive.html` | Full archive grouped by year. |
| `.github/workflows/jekyll.yml` | CI build + deploy. |

Migration scripts live outside this repo in `../migrate/`.

## Local preview

Modern Ruby:

```sh
bundle install
bundle exec jekyll serve
```

Or, without Ruby, use the Python static preview generated by the migration
tooling (`../migrate/preview.py`) which opens `_preview/index.html` in a
browser.

## Comments

Historical comments are baked into each post as read-only static HTML. New
comments use [giscus](https://giscus.app): enable repo **Discussions**, register
giscus, then fill the `giscus.*` keys in `_config.yml` and set `enabled: true`.
